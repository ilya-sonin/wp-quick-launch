# Быстрый старт

## 1. Первый запуск

```bash
./start.sh
```

Ответьте на вопросы:
- Порт WordPress (например: 8082)
- Порт phpMyAdmin (например: 8083)  
- Порт MySQL (например: 3337)
- Имя проекта (например: my-project)

## 2. Откройте браузер

WordPress: http://localhost:ВАШІ_ПОРТ

Следуйте мастеру установки WordPress.

## 3. Управление

**Остановка:**
```bash
./stop.sh
```

**Повторный запуск:**
```bash
./start.sh
```
(при повторном запуске настройки не спрашиваются)

## Структура

- `wordpress/` - все файлы WordPress
- `wordpress/wp-content/themes/` - ваши темы
- `wordpress/wp-content/plugins/` - ваши плагины

## Полезное

**Логи:**
```bash
docker-compose logs -f wordpress
```

**Полная очистка (удалит все данные!):**
```bash
./stop.sh
# Выберите пункт 2
```

**Сменить порты:**
```bash
./stop.sh
# Выберите пункт 3
# Затем запустите ./start.sh снова
```
